# 🧠 Wonder – Projet Symfony

Bienvenue dans **Wonder**, un projet Symfony réalisé dans le cadre de ma formation de développeur web PHP/Symfony. 

L'objectif ? Créer une application sociale inspirée de **Quora**, où les utilisateurs peuvent poser des questions, y répondre, et interagir entre eux.

## 🎯 Objectifs pédagogiques

Ce projet a pour but de mettre en pratique les compétences acquises tout au long de la formation.  
Il utilise un grand nombre de composants Symfony et outils modernes du développement web.

### 🛠️ Technologies et outils utilisés

- Symfony (v6+)
- Twig
- Composer
- Webpack Encore
- Mailer
- Form
- Doctrine ORM
- Bootstrap 5
- Font Awesome
- NGINX + PHP-FPM (pour le déploiement)
- et bien d'autres bundles et composants Symfony !

---

## 📦 Versions & Avancement

Le développement est découpé en plusieurs **versions** progressives :

| Version | Contenu                                                                 |
|---------|-------------------------------------------------------------------------|
| v1 - version-1      | 🔧 Mise en place du layout global : `Header`, `Footer`, `Sidebar`      |
| v2 - version-2      | 📝 Création et intégration des **formulaires** avec le composant Form |
| v3 - version-3      | 🗄️ Configuration et intégration de la **base de données** avec Doctrine |
| v4 - version-4      | 🔐 Mise en place du système d’**authentification** (security/login)    |

> Le projet est en cours d’évolution 🚀

---

## 🚀 Déploiement

L’application sera déployée en production avec :
- **NGINX** comme serveur web
- **PHP-FPM** pour exécuter le backend Symfony
